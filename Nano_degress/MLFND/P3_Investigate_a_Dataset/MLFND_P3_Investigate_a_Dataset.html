def plotbar(data1, data2, title, c1, c2, n, l1, l2):
    n_groups = 1
    means_men = (data1)
    means_women = (data2)

    index = np.arange(n_groups)
    bar_width = 0.15

    opacity = 0.7
    error_config = {'ecolor': '0.3'}

    rects1 = plt.bar(index, means_men, bar_width,
                     alpha=opacity,
                     color=c1,
                     yerr=0,
                     error_kw=error_config,
                     label=l1)

    rects2 = plt.bar(index + 0.5 + bar_width, means_women, bar_width,
                     alpha=opacity,
                     color=c2,
                     yerr=0,
                     error_kw=error_config,
                     label=l2)

    plt.xlabel('')
    plt.ylabel('Percentages')
    plt.title(title)
    plt.xticks(index + bar_width, (''))
    plt.legend()
    plt.yticks(n)
    plt.legend(loc='upper left')
    plt.tight_layout()
    plt.show()

'''
This method maps 0's and 1's in the Survived
column to False and True respectively.
'''


def convert_survived(data):
    data_values = {1: True, 0: False}
    data['Survived'] = data['Survived'].map(data_values)
    return data

Filters passengers based on provided parameters.
Returns a dataframe representing the filtered
passenger data.
'''


def filter_passengers(key1, key2, key1_value, key2_value):
    return titanic_df[(titanic_df[key1] == key1_value) &
                      (titanic_df[key2] == key2_value)]

Documentation for <a href = "https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.pearsonr.html" style = "text-decoration: none">pearson's r</a> states: Like other correlation coefficients, this one varies between -1 and +1 with 0 implying no correlation. Correlations of -1 or +1 imply an exact linear relationship. Positive correlations imply that as x increases, so does y. Negative correlations imply that as x increases, y decreases.

Inference: r is negative for x representing passenger class and y representing survived column. Higher the class number lower is the chance of survival. We should be careful here to note that higher class number means in descending order (3 = lower, 2 = middle and 1 = upper). This result is in agreement with our analysis of relationship between class and survival in Revisiting the Questions section.

4. Conclusion¶

All statistical concclusions are explained within the Inference part of each section.

Summary:

  • The entire analysis and exploration was mainly about survival of passengers aboard Titanic ship.
  • There are other things we could have investigated independently such as passenger demographics viz, gender, age and class.
  • But demographics alone is'nt much useful for practical insights into the data.
  • Hence within the analysis process the main factor we were interested was survival of each pasenger.
  • All demographics were related to survival rate and explored from different angles.

Limitations in the data:

  1. The data wrangling phase showed that there were large gaps in data, specially when it came to the age of passengers.

  2. Two embarkation ports were missing.
    3. </ol>
    Here is how missing data was dealth with:
    Missing ages: We used imputed values based on passenger class and age.
    Missing embarkation ports:Although one can try to correlate embarkation port and
    survival, in our analysis we have not done so for simplicity. That is we have considered
    variables which were highly correlated to survival.

    Correlation does not imply causation. But finding out correlation gives us a general idea of how consistent are our analysis results. It seems age, class, ticket fare and gender are highly correlated. As per project rubric only three variables i.e. age, class and gender were considered for the analysis process.

    There are other limitations on the data such as:

    1. Whether lurking variables were constant or kept consistent. Example availability of life boats.

    2. The impact of size of sample data.

    3. One more categorization that could have been included in the data was whether person was a crew or passenger.
